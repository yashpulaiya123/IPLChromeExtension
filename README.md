# IPLChromeExtension

#### This is a Chrome Extension project built from scratch leveraging HTML, CSS, Javascript & JSON Files, used to check live IPL cricket match scores by fetching reliable data through integrated APIs on just a single click to be presentable on screen to facilitate ease for checking scores even while working without opening a new tab and typing just to check live scores in every few minutes.
